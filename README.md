# jewellery-inventory-manager

A web application to manage inventory for a jewellery business

## Overview

This project was generated by AIxNative and contains the following modules:

- **Database Schema** (database): Prisma schema with Item, Supplier, Purchase, Sale, Customer, and User models
- **Authentication** (auth): NextAuth authentication for users with roles
- **Item Service** (service): Business logic for items
- **Supplier Service** (service): Business logic for suppliers
- **Purchase Service** (service): Business logic for purchases
- **Sale Service** (service): Business logic for sales
- **Customer Service** (service): Business logic for customers
- **Metal Rate Service** (service): Business logic for metal rates
- **Price Calculation Service** (service): Business logic for price calculations
- **Report Service** (service): Business logic for reports
- **Dashboard Service** (service): Business logic for dashboard
- **API** (api): REST endpoints for services
- **UI** (ui): React frontend for the application

## Getting Started

## Modules


### Database Schema

**Type:** database

Prisma schema with Item, Supplier, Purchase, Sale, Customer, and User models

**Acceptance Criteria:**
- Item model with id, name, category, metalType, purity, grossWeight, netWeight, stoneDetails, makingCharges, wastage, finalPrice, and timestamps
- Supplier model with id, name, address, and contactDetails
- Purchase model with id, supplierId, itemId, quantity, and date
- Sale model with id, customerId, itemId, quantity, and date
- Customer model with id, name, address, and contactDetails
- User model with id, name, email, password, role, and timestamps
- Foreign key relationships between models
